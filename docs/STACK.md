# Stack technique - Swing Digital

## Frontend

| Technologie | Usage |
|-------------|-------|
| HTML5 | Structure semantique |
| CSS3 | Mise en page, animations, responsive |
| JavaScript (vanilla) | Interactions, scroll, navigation |

## Approche

- **Pas de framework** : site vitrine statique, pas besoin de React/Vue/etc.
- **Mobile-first** : design responsive, priorite mobile
- **Performance** : images optimisees, CSS/JS minimises
- **Accessibilite** : structure semantique, alt texts, contrastes

## Polices

```
Brandon Black      -> Titres principaux
Brandon Regular    -> Corps de texte
Fragen Bold        -> Accents visuels
Raleway Bold       -> Navigation
Roboto Bold        -> Elements UI
Walden Black       -> Elements decoratifs
```

## Hebergement

A definir (Netlify, Vercel, OVH, ou autre).

## Outils de developpement

- Pas de bundler (Webpack, Vite, etc.)
- Optimisation images : ImageOptim ou squoosh.app
- Validation : W3C Validator
- Tests responsive : DevTools navigateur
